# Настройка Kinescope.io Integration

## Быстрый старт

Интеграция Kinescope не требует настройки API ключей или переменных окружения. Просто используйте Video ID из Kinescope.

## Как получить Kinescope Video ID

1. Загрузите видео в [Kinescope](https://kinescope.io/admin)
2. В списке видео нажмите на нужное видео
3. В URL или в настройках видео найдите ID видео
4. ID выглядит как: `98a33745-5dd5-4ff0-a203-d9edf1581f5b` (UUID формат)
5. Используйте этот ID в админ-панели при создании урока

## Использование

1. В админ-панели откройте раздел "Уроки"
2. При создании/редактировании урока введите Kinescope Video ID
3. Видео автоматически появится на странице урока

## Формат embed

Система использует прямой embed от Kinescope:
```
https://kinescope.io/embed/{videoId}
```

Никаких дополнительных настроек не требуется!

## Устранение проблем

### Видео не отображается
- Проверьте правильность Video ID в админ-панели
- Убедитесь, что видео существует в Kinescope
- Проверьте статус видео в панели Kinescope (должен быть "ready")
- Убедитесь, что домен добавлен в настройки проекта Kinescope (если требуется)

### Ошибка "Video ID не указан"
- Убедитесь, что в админ-панели заполнено поле "Kinescope Video ID"
- Проверьте, что ID скопирован полностью (обычно это UUID)

### Видео не воспроизводится
- Проверьте консоль браузера на наличие ошибок
- Убедитесь, что видео обработано в Kinescope (статус "ready")
- Проверьте настройки приватности видео в Kinescope
